Comment utiliser dans Grist
1. Importez vos 3 fichiers Excel via Ajouter nouveau > Importer depuis un fichier.
2. Créez une nouvelle table vide ou utilisez une table existante.
3. Créez une colonne de formule et collez la formule générée ci-dessus.
4. Grist utilise Python — les jointures se font via VLOOKUP ou via des références de colonnes.
